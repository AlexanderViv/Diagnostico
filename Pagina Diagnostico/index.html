WWW

WWW son las siglas de World Wide Web, que en español significa red informática mundial. Es un sistema de páginas web interconectadas que se pueden acceder a través de internet. La WWW es una aplicación que se construye sobre internet, por lo que no son lo mismo.

Internet

Internet es una red de computadoras interconectadas a nivel mundial que permite el intercambio de información. Es una herramienta que facilita el trabajo, el aprendizaje, el entretenimiento y la comunicación.

Servidor

Un servidor es un sistema que proporciona recursos, datos, servicios o programas a otros ordenadores, conocidos como clientes, a través de una red. En teoría, se consideran servidores aquellos ordenadores que comparten recursos con máquinas cliente.

Centro de datos

Un centro de datos es una instalación física que alberga los sistemas de computación y los datos de una organización. Su función es almacenar, procesar, organizar, asegurar y conservar la información de la empresa.

Página Web

Una página web es un documento electrónico que se encuentra en la World Wide Web y que contiene información visual y organizada. Se puede acceder a ella a través de internet y se muestra en un navegador web.

Sitio Web

Un sitio web es un conjunto de páginas web que están interconectadas y que se pueden acceder a través de internet. Los sitios web pueden contener información, imágenes, videos, y otros elementos multimedia.

IP

IP significa “protocolo de Internet”, que es el conjunto de reglas que rigen el formato de los datos enviados a través de Internet o la red local. En esencia, las direcciones IP son el identificador que permite el envío de información entre dispositivos en una red.

ICANN

ICANN es una entidad internacional sin ánimo de lucro responsable de alojar direcciones IP, gestionar dominios genéricos y territoriales, y asignar identificadores de protocolos, como los certificados SSL.

HTTP

HTTP son las siglas en inglés de Hypertext Transfer Protocol, que en español significa Protocolo de Transferencia de Hipertexto. Es un protocolo que permite la comunicación entre navegadores y servidores web. El HTTP es la base de la transmisión de datos en la web.

HTML

HTML (Hypertext Markup Language) es un lenguaje de programación que se utiliza para crear y estructurar páginas web. Es el código que define la estructura de los contenidos de una página web, como imágenes, texto, enlaces y multimedia.

Navegador Web

Un navegador web es un programa que permite acceder a la World Wide Web y navegar por las páginas web. Los navegadores interpretan el código de las páginas web y lo muestran en pantalla.

CSS

El CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación visual de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML.

JavaScript

JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas. Se usa para controlar multimedia, animar imágenes, actualizar fuentes de redes sociales, y más.

Front-end

El front-end es la parte de una página web o aplicación que los usuarios pueden ver e interactuar. En el desarrollo web, el front-end es todo lo que se refiere a la presentación y estilizado del contenido.

Back-end

El back end es la parte de un sitio web o aplicación que se encarga de la lógica y el procesamiento de datos. Es la parte invisible que se encuentra detrás de la interfaz de usuario.

URL

Una URL (Uniform Resource Locator) es una dirección única que identifica un recurso en internet, como una página web, imagen, archivo de audio o video.

DNS

DNS son las siglas en inglés de Domain Name System, que en español significa "Sistema de nombres de dominio". Es un sistema que traduce los nombres de dominio a direcciones IP, lo que permite a los navegadores cargar páginas web.

WEB 1.0

La Web 1.0 fue la primera etapa de la World Wide Web (WWW). Se caracterizó por ser una web estática, con páginas web que no permitían la interacción entre el usuario y el contenido.

WEB 2.0

La Web 2.0 es una generación de Internet que se caracteriza por la colaboración entre los usuarios y el intercambio de información.

WEB 3.0

La Web 3.0, también conocida como Web3, es la tercera generación de la World Wide Web. Se trata de un conjunto de tecnologías que buscan democratizar el entorno digital y descentralizar el control de los datos.

WEB 4.0

Web 4.0 es una capa de integración necesaria para la explotación de la Web semántica. Es un nuevo modelo de Internet que nace con el objetivo de resolver las limitaciones de la red a día de hoy.

WEB 5.0

La Web 5.0 es una visión de la web que se caracteriza por ser descentralizada, sensorial y emotiva. Esta versión de la web busca mejorar la experiencia del usuario a través de la inteligencia artificial, la realidad virtual y aumentada.
